# Knowing the Name of Something vs Knowing Something

## About

A concise distillation of Feynman's distinction between naming and understanding. Knowing every name for a bird in every language tells you nothing about the bird. The test: can you explain the concept without using the newly learned word? If not, you've memorized a label, not internalized knowledge. Feynman's method: "Take apart the toy; see how it works."
